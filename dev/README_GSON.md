# Téléchargement

(En cas de perte du dossier lib et de son JAR) :

Télécharger 'gson-2.8.9.jar' : https://repo1.maven.org/maven2/com/google/code/gson/gson/2.8.9/gson-2.8.9.jar


# Construction du projet

1) Ajout du dossier tdtp dans le dossier src.

2) Ajouter le dossier lib dans le projet (New > Folder)

3) Dans lib copiez : gson-2.8.9.jar

4) Niveau projet éclipse : - Faites un clic droit sur le projet → Build Path → Configure Build Path → Libraries → Add JARs...
   - Sélectionnez `lib/gson-2.8.9.jar` et validez.

5) Lancer l'application (./src/tdtp/ApplicationTPTD.java)
